  function middleware(req: NextRequestWithAuth) {

**`src/app/api/admin/users/route.ts`** — GET list + POST create:

GET handler:
1. `getServerSession(authOptions)` — 401 if null
2. Check `ROLE_PERMISSIONS[activeRole].canManageUsers` — 403 if false (where `activeRole = (session.user as any).activeRole ?? session.user.role`)
3. `prisma.user.findMany({ include: { userLocations: { include: { location: { select: { id: true, name: true } } } } }, orderBy: { name: 'asc' } })`
4. Map results to exclude `password` field from response
5. Return 200

POST handler:
1. Session + canManageUsers check
2. Zod schema:
```typescript
z.object({
  name: z.string().min(1, "Name is required").max(100),
  email: z.string().email("Invalid email format"),
  password: z.string().min(8, "Password must be at least 8 characters"),
  role: z.enum(['EMPLOYEE', 'MANAGER', 'ADMIN']),
  locationId: z.string().optional(),
  locationIds: z.array(z.string()).optional(),
})
```
3. Check email uniqueness: `prisma.user.findUnique({ where: { email: email.toLowerCase() } })` — 409 if exists
4. Hash password: `import { hash } from 'bcryptjs'` — `const hashedPassword = await hash(password, 12)`
5. Create user with transaction:
```typescript
await prisma.$transaction(async (tx) => {
  const user = await tx.user.create({
    data: {
      name,
      email: email.toLowerCase(),
      password: hashedPassword,
      role,
      forcePasswordReset: true,
      locationId: role !== 'EMPLOYEE' ? locationId : null,
    }
  })
  if (role === 'EMPLOYEE' && locationIds?.length) {
    await tx.userLocation.createMany({
      data: locationIds.map(locId => ({ userId: user.id, locationId: locId }))
    })
  }
  return user
})
```
6. Return 201 (exclude password from response)
